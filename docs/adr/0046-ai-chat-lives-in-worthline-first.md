# AI chat lives in worthline first

The first AI experience is an in-app chat that reads the workspace through the existing **agent view**. MCP remains an external channel for third-party agents to inspect the same facts, but it is not the primary user experience for worthline's own **financial assistant**.

This keeps the assistant close to the product's scopes, labels, previews, confirmation flows, and future **assistant proposals**. It also avoids making Claude, ChatGPT, or another external MCP client the canonical UI for a feature that needs worthline-specific interaction, validation, and audit context.
