# AI is a financial assistant, not a regulated advisor

worthline's AI experience may explain the user's financial position, analyze trade-offs, and recommend actions, but it is framed as a **financial assistant**, not a regulated advisor or automatic manager. Recommendations must expose the facts and assumptions they rely on, and any mutation still flows through a confirmed **assistant proposal** rather than direct execution.

This preserves the useful product behavior the user wants while keeping the system boundary honest: worthline can help reason over the workspace, but it does not pretend the model output is a licensed advisory service or a permission to bypass validation, preview, audit, and user confirmation.
