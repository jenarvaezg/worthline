# The financial assistant surfaces internal sources

worthline's **financial assistant** shows the internal facts it consulted in a compact way, such as current portfolio, monthly history, a specific holding detail, debt facts, FIRE progress, or data-quality signals. These are workspace sources, not web citations: they tell the user which worthline facts grounded the answer.

This makes recommendations and explanations auditable enough for personal finance. Tool activity should be visible without dumping raw logs, so the user can distinguish grounded analysis from model fluency and can spot when the assistant answered from the wrong scope or stale fact.
